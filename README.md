# Don't Share - Очистка текста и PDF от личных данных

Локальное веб-приложение на Python для очистки текстовых и PDF файлов от персональных данных (PII - ФИО, номера телефонов, адреса электронной почты, локации) с использованием Microsoft Presidio.

---

## ✨ Возможности

* **Drag-and-Drop загрузка**: Быстрая загрузка файлов формата `.txt` и `.pdf`.
* **Интерактивный предпросмотр**:
  * Для текста: встроенный текстовый редактор с подсветкой сущностей.
  * Для PDF: постраничный рендеринг документа в изображения с интерактивной наложенной разметкой PII.
* **Гибкое управление PII**:
  * Клик на рамку/выделение для исключения сущности из удаления (подсвечивается зеленым цветом).
  * **Правый клик мыши (ПКМ)** по рамке на PDF для её мгновенного удаления.
  * Выделение текста вручную для добавления масок (режим TXT).
  * Интуитивное **рисование рамок выделения** прямо на странице PDF зажатием левой кнопки мыши.
* **Экспорт очищенного файла**: Скачивание чистого TXT-файла с тегами-заглушками (`<PERSON>`) или PDF-документа с наложенными черными плашками.
* **Поддержка языков**: Интегрированные модели NLP для русского (`ru_core_news_sm`) и английского (`en_core_web_sm`) языков.

## 🛠️ Запуск из исходного кода

### 1. Установка внешних утилит (Windows)

Для работы с PDF приложению требуются:
1. **Tesseract OCR** (распознавание текста на картинках/сканах):
   * Скачайте установщик для Windows (например, с github.com/UB-Mannheim/tesseract/wiki).
   * Установите в путь по умолчанию: `C:\Program Files\Tesseract-OCR`.
2. **Poppler** (преобразование страниц PDF в изображения):
   * Скачайте бинарные файлы Windows (например, с github.com/oschwartz10612/poppler-windows/releases).
   * Распакуйте архив и запомните путь к папке `bin` (например, `C:\poppler\poppler-26.02.0\Library\bin`).

### 2. Установка зависимостей и запуск

Убедитесь, что у вас установлен Python (рекомендуется 3.10+):

```powershell
# Установка библиотек
pip install --ignore-requires-python -r requirements.txt

# Скачивание языковых моделей spaCy
python -m spacy download en_core_web_sm
python -m spacy download ru_core_news_sm

# Запуск приложения
.\run.ps1
```

После старта откройте в браузере: **[http://127.0.0.1:8000](http://127.0.0.1:8000)**.
В правом верхнем углу интерфейса нажмите на иконку шестеренки и введите путь к вашей папке `bin` у Poppler. Настройки сохранятся в файл `.env` навсегда.

## 📦 Сборка в автономный EXE (Портативная версия)

Для компиляции приложения в один исполняемый файл `.exe` используется библиотека **PyInstaller**:

1. Запустите скрипт автоматической сборки:
   ```powershell
   .\build_exe.ps1
   ```
   Этот скрипт установит PyInstaller и скомпилирует файл `dont_share.exe` в папке `dist/`.

2. Для создания полностью независимой (portable) версии:
   * Создайте новую папку, например, `DontShare_Portable`.
   * Скопируйте туда созданный файл `dont_share.exe` из папки `dist`.
   * Создайте в этой же папке подкаталог `tesseract` и скопируйте туда все установленные файлы Tesseract OCR (так, чтобы файл `tesseract.exe` лежал по пути `tesseract/tesseract.exe`).
   * Создайте подкаталог `poppler` и скопируйте туда распакованный Poppler (так, чтобы исполняемый файл `pdftoppm.exe` лежал по пути `poppler/Library/bin/pdftoppm.exe` или `poppler/bin/pdftoppm.exe`).
   
При таком размещении приложение на другом ПК запустится и сразу определит статус утилит как `Активен` без какой-либо настройки!

## 💾 Создание инсталлятора (Setup.exe)

Для сборки полноценного установщика для Windows используется бесплатный инструмент **Inno Setup**:

1. Скачайте и установите **Inno Setup** (jrsoftware.org/isinfo.php).
2. Поместите папки `tesseract` и `poppler` в каталог проекта, чтобы они были готовы к упаковке.
3. Откройте файл `dont_share.iss` в Inno Setup Compiler.
4. Нажмите кнопку **Compile** (или нажмите `F9`).
5. В папке `Output/` сгенерируется файл установщика `dont_share_setup.exe`. Он установит программу в систему, скопирует все бинарные зависимости Poppler/Tesseract, настроит пути и создаст ярлык на Рабочем столе.
